#### 定义

ES6 允许按照一定模式，从数组和对象中提取值，对变量进行赋值，这被称为解构

#### 示例摘要

``` 
let [a, b, c] = [1, 2, 3]; 

let [head, ...tail] = [1, 2, 3, 4];
head // 1
tail // [2, 3, 4]

let [x, y, ...z] = ['a'];
x // "a"
y // undefined
z // []
```
